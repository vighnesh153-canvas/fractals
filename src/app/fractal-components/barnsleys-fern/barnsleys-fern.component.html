Barnsley's Fern is a fractal that is generated using 4 functions. Cumulative probability is used to determine which function should transform the old coordinates to new ones. The initial values of x and y are 0. The transformation functions are as follows:

f1 is chosen 1% of the time:
xn + 1 = 0
yn + 1 = 0.16yn

f2 is chosen 85% of the time:
xn + 1 = 0.85xn + 0.04yn
yn + 1 = -0.04xn + 0.85yn + 1.6

f3 is chosen 7% of the time:
xn + 1 = 0.2xn - 0.26yn
yn + 1 = 0.23xn + 0.22yn + 1.6

f4 is chosen 7% of the time:
xn + 1 = -0.15xn + 0.28yn
yn + 1 = 0.26xn + 0.24yn + 0.44
